# Blog

一个基于 Next.js 的个人博客与后台管理系统。

## 线上地址

- https://blog-fhrd.vercel.app/

## 功能特性

- 首页、博客列表、详情页、分类页、标签页、关于页
- Markdown 写作（ByteMD + GFM + 代码高亮）
- 后台管理：文章/分类/标签 CRUD
- 图片上传（Vercel Blob）
- 登录鉴权（Better Auth + Prisma）
- 明暗主题切换

## 技术栈

- Next.js 16 + React 19 + TypeScript
- Prisma + PostgreSQL
- Better Auth
- Tailwind CSS
- ByteMD
- SWR

## 本地开发

```bash
npm install
npm run dev
```
